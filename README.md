#  Banking Analysis Project

##  Project Title
**Banking-Analysis**

##  Description
The **Banking Analysis Project** focuses on analyzing banking data to extract meaningful insights related to customers, advisors, investments, relationships, and demographics.  
This project uses **Python, Excel, CSV datasets, Power BI, and PowerPoint** to perform data cleaning, analysis, visualization, and reporting.

The goal is to help understand customer behavior, advisor performance, and relationship patterns in the banking sector using data-driven techniques.

---

##  Objectives
- Analyze customer and banking datasets
- Identify patterns and trends in banking relationships
- Visualize insights using Power BI dashboards
- Perform exploratory data analysis using Python
- Present findings in a structured and professional format
